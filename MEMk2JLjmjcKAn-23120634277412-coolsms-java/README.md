## 실제 개발연동 시 추가해야 할 의존성  

### Maven
```xml
<dependency>
    <groupId>net.nurigo</groupId>
    <artifactId>sdk</artifactId>
    <version>4.2.7</version>
</dependency>
```

### Gradle
```groovy
implementation 'net.nurigo:sdk:4.2.7'
```

### Gradle(kotlin)
```kotlin
implementation("net.nurigo:sdk:4.2.7")
```
